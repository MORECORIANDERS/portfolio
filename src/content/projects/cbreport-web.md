## 概述

可转债日报（CB Report）是一个每日自动更新的可转债市场数据看板。它在每个工作日收盘后自动采集、计算并发布一份完整的市场日报，覆盖 300 余只可转债的行情、分布与榜单，让研究者和交易者在一屏之内把握当日全貌。

## 问题

可转债市场标的众多、行情分散，逐只翻看耗时且难以建立整体认知。现有的看盘工具要么信息过载、要么更新滞后，缺少一份「每天看一眼就够」的精炼日报。我想要的，是闭市后自动生成、结构稳定、可回溯的每日快照。

## 方案

- **全自动数据管道**：CloudBase 云函数在每个工作日 15:15（北京时间）自动触发，完成数据采集、清洗、指标计算与静态页面生成，全程无人值守。
- **结构化日报**：固定六大板块 —— 市场概况、价格分布、行业分布、涨幅 TOP10、成交额 TOP10、跌幅 TOP10，每次打开都是熟悉的阅读路径。
- **中位数驱动**：用中位数而非均值呈现涨跌与成交，避免极端样本扭曲对市场温度的判断。
- **静态交付**：最终产物是纯静态页面，部署在 GitHub Pages，加载快、可归档、零运维成本。

## 成果

站点已稳定运行，每日工作日按时自动更新，覆盖 304 只可转债。对我个人而言，它把过去需要半小时翻盘的流程压缩成了「打开看 3 分钟」；也成了几个朋友每天必看的市场温度计。
